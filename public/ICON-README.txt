# Icon/Favicon

Du måste skapa en icon för sajten:

## Fil som behövs:
- **app/icon.png** - 512x512px PNG med transparent bakgrund

## Rekommendationer:
- Använd ditt företags logotyp eller ett symbol som representerar andlighet/spådomar
- Enkel design som fungerar i liten storlek
- Transparent bakgrund
- Format: PNG
- Storlek: 512x512px (Next.js kommer automatiskt att generera andra storlekar)

## Verktyg för att skapa ikoner:
- **Canva** - Enkel online-designer
- **Figma** - Professionell design
- **Adobe Illustrator** - För vektorlogotyper
- **Favicon.io** - Generera favicon från text eller bild

Next.js App Router kräver att ikonen placeras som `app/icon.png` för att automatiskt genereras till olika format.
